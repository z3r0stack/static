Git Step by Step

git config



$ git config --global user.name 'Name Surname'
$ git config --global user.email 'name@example.com'

create directory





$ mkdir begin
$ cd begin 
$ git init 
Initialized empty Git repository in ~/begin/.git/

create file begin.txt




$ echo 'let begin' > begin.txt
$ cat begin.txt
let begin

Git add file to git




$ git add begin.txt
warning: LF will be replaced by CRLF in begin.txt.
The file will have its original line endings in your working directory.

Git Check status










$ git status

        new file:   begin.txt

Git commit




$ git commit -m 'add begin'

warning: LF will be replaced by CRLF in begin.txt.

Check status add




$ git status
On branch master
nothing to commit, working directory clean

$ echo 'line second' >> begin.txt
$ echo 'git md' > README.md

Create Branch

$ git branch hophop

Check Branch




$ git branch
  hophop
* master

Switch to branch hophop



$ git checkout hophop
Switched to branch 'hophop'

Edit file and add new



$ echo 'testt' >> begin.txt
$ echo 'dev' >> def.txt

Check status new
















$ git status

  (use "git checkout -- <file>..." to discard changes in working directory)

Clean file git



$ git clean -f

$git commit -am 'update file'

Delete branch





$ git checkout master
